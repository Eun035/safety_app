# C-Safe (C-PM Safety Platform)

**"안전을 넘어서, 스마트한 PM 문화를 위한 데이터 기반 리워드 솔루션"**

C-Safe는 퍼스널 모빌리티(PM) 이용자의 안전을 보장하고, 올바른 주차 문화를 조성하기 위해 개발된 엣지 AI 기반 안전 플랫폼입니다. 단순히 포인트 보상을 주는 앱을 넘어, **헬멧 착용 및 합법 구역 주차라는 사용자 행동을 '안전 데이터'로 자산화**하여 지역 사회의 안전사고를 실질적으로 저감하는 B2G/O2O 핵심 비즈니스 로직을 구현합니다.

---

## 🛡️ 핵심 가치: 왜 C-Safe인가?

1.  **실질적 사고 저감**: Edge AI 헬멧 인증을 통해 탑승 전 안전 장구 착용을 제도적으로 유도하여 사고 시 중증 발생률을 획득된 데이터를 통해 낮춥니다.
2.  **데이터 기반 안전 자산화**: 개별 사용자의 안전 운행 습관(급감속, 주행 안정도, 헬멧 착용 여부)이 데이터베이스화되어, 추후 지역 교통 인프라 개선을 위한 정책적 근거 자료(Data Lake)로 활용됩니다.
3.  **지역 사회 상생 (O2O)**: 안전 데이터를 발생시킨 사용자에게 지역 화폐(천안사랑카드) 포인트를 지급함으로써, '안전 행위 = 경제적 가치'라는 선순환 구조를 구축합니다.

## 🚀 주요 기능 및 핵심 로직 상세

### 1. 지능형 안전 온보딩 (Safety Compliance)
- **다국어 지원**: 외국인 유입이 많은 지역적 특성을 반영하여 i18next 기반 한국어/영어 전환 시스템 구축.
- **안전 교육 퀴즈**: 전체 9문항 중 3문항을 무작위 추출하여 제공. '안전 규정 숙지' 없이는 서비스 진입이 불가능하도록 설계되어 사고 예방 의식을 고취합니다.

### 2. Edge AI 실시간 헬멧 검증 (On-Device Inference)
- **로직**: 서버로 이미지를 전송하지 않고 사용자의 기기(Edge) 내에서 `OpenVINO/ONNX Runtime`을 활용해 실시간으로 헬멧 착용 여부를 판별합니다.
- **안전 인증 프로세스**: 
    - 카메라 가이드라인 내에 얼굴과 헬멧이 인식되면 **2초간 누적 검증** 진행.
    - 검증 성공 시에만 운행 시작 버튼 활성화.
    - 개인정보 보호와 실시간성을 동시에 확보하며, 헬멧 미착용으로 인한 안전사고를 원천 차단합니다.

### 3. 하버사인 지오펜싱 기반 정밀 주차 검증
- **로직**: 운행 종료 시점의 사용자 GPS 좌표와 천안시 공공 데이터로 등록된 410여 개의 PM 주차 구역 간 직선 거리를 **하버사인 공식**으로 계산합니다.
- **판별 기준**:
    - **반경 10m 이내**: 합법 주차로 자동 검증 -> 100P 리워드 즉시 지급.
    - **반경 10m 초과**: 불법 주차 경고 및 음성 안내 사운드 출력.
- **가치**: 무단 방치로 인한 보행자 사고를 미연에 방지하고 깨끗한 거리 환경을 조성합니다.

### 4. 실시간 주행 텔레메트리 (Safe-Driving Telemetry)
- **급브레이크 감지**: 주행 중 발생하는 가감속 데이터를 분석하여 급제동 횟수를 기록합니다.
- **주행 안정도(Stability)**: 기기 흔들림을 측정하여 40% 미만 시 "감속 운행" 음성 가이드를 즉시 송출합니다.
- **결과 데이터**: 이 데이터는 'Safety Score'로 환산되어 사용자의 안전 주행 지표로 관리됩니다.

### 5. 에코 배지 (Eco-Engagement)
- **탄소 절감량 계산**: 주행 거리 대비 탄소 저감량(kg CO2)을 실시간 산출.
- **보상**: 일정 수치 달성 시 '에코 배지'를 수여하고 SNS 공유를 통해 지역 안전 문화를 전파합니다.

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: React 19, Vite 6 (고성능 SPA 최적화)
- **Styling**: Tailwind CSS 4 (Cyberpunk Aesthetic 디자인)
- **State Management**: Zustand, useLocalStorage (사용자 상태 보존)
- **Map Integration**: React Kakao Maps SDK (지자체 공공데이터 시각화)
- **Database/Auth**: Supabase (실시간 안전 로그 및 익명 인증)
- **Intelligent Logic**: 
    - `haversine-distance`: 지오펜싱 정밀 검사
    - `javascript-obfuscator`: 핵심 비즈니스 로직(포인트 지급 등) 보호

## 📂 프로젝트 아키텍처 및 로직 파일

- `src/App.jsx`: 전체 라이드 세션 및 전역 상태 관리 (O2O 포인트 연동).
- `src/components/common/HelmetDetectionCamera.jsx`: Edge AI 카메라 제어 및 UI 가이드.
- `src/hooks/useEdgeAI.js`: 실제 딥러닝 모델 기반 추론 처리 및 진행도 관리.
- `src/utils/distance.js`: 하버사인 거리 계산 알고리즘.
- `src/data/pm_parking_data.json`: 천안시 PM 주차 구역 410개소 정제 데이터.
- `vite.config.js`: 보안을 위한 빌드 타임 코드 난독화 설정.

## 📈 비즈니스 확장성

1.  **B2G 대시보드 연동**: 축적된 위험 구간 및 안전 데이터를 시청에 제공하여 인프라 개선 근거로 활용.
2.  **보험사 제휴**: 헬멧 검증 및 안전 주행 데이터를 기반으로 한 PM 전용 보험 상품 연계.
3.  **플랫폼 고도화**: AR(증강현실) 뷰를 통한 가상 주차 가이드 기능 확장성 확보.
