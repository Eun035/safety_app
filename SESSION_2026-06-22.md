## 6. 신규 기능 동작 요약

### 6.1 ShareCard (인스타 스토리 1080×1920 카드)
- 매트 블랙 + `#CCFF00` 네온 그린 액센트
- 동적 타이틀: `"{routeLabel} {distance.toFixed(1)}km 비행 완료 ⚡"`
- 3행 지표 (모노스페이스): DURATION (MM:SS) / AVG SPEED (km/h) / SAFETY SCORE (%)
- 안전 스코어 = `max(60, 100 - suddenBrakeCount × 5)` (UI 전용 계산)
- `helmetOn=true` 시 `⛑️ SAFE RIDER` 배지 + 네온 글로우
- SVG 네온 경로 + 그리드 패턴 배경, html-to-image의 `toPng`로 캡처

### 6.2 Wake Lock (라이딩 중 화면 절전 차단)
- `useWakeLock(isRiding)` 호출 — `isRiding && isVisible`일 때만 `navigator.wakeLock.request('screen')`
- 백그라운드 진입 시 브라우저가 자동 해제, 복귀 시 `usePageVisibility` 의존성으로 effect 재실행 → 자동 재획득
- 미지원 브라우저(일부 iOS WebView) silent no-op

### 6.3 안전 퀴즈 보이스 개선
- `QUIZ` 프리셋(rate 0.85, pitch 0.95): 차분하고 또렷한 낭독
- `QUIZ_FEEDBACK` 프리셋(rate 1.0, pitch 1.05, interrupt:true): 즉답
- 문제 낭독 중 답을 누르면 즉시 낭독 중단 + "정답!" / "오답." 발화 + 1.5초 뒤 다음 문제
