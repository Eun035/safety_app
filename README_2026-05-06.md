# C-Safe (천안 캠퍼스 PM 안전 주행 가이드) - 프로젝트 현황 (2026-05-06)

본 문서는 2026년 5월 6일 기준, **C-Safe** 애플리케이션의 개발 진척도와 데이터 연동 상태를 요약한 보고서입니다.

## 🚀 프로젝트 개요
천안 지역(특히 대학가) 내 전동 킥보드(PM) 이용자를 위한 안전 주행 및 합법 주차 가이드 서비스입니다. 사이버펑크 스타일의 UI와 실시간 데이터를 결합하여 사용자에게 안전한 경로와 보상을 제공합니다.

## 📊 실제 데이터 연동 현황 (Real Data Status)
본 프로젝트는 정적 목 데이터를 배제하고 다음과 같은 실제 데이터를 기반으로 작동합니다.

1.  **실시간 위치 추적 (Browser Geolocation API)**
    *   사용자의 현재 좌표(위도, 경도) 및 주행 속도를 실시간으로 수신합니다.
    *   주행 중 이동 경로를 추적하여 지도상에 경로(Polyline)를 생성합니다.

2.  **실시간 안전 데이터 (Supabase Realtime)**
    *   **사고/위험 제보**: 사용자가 등록한 위험 구역 정보를 Supabase DB에서 가져오며, 새로운 제보 발생 시 실시간(Realtime)으로 모든 사용자에게 전파됩니다.
    *   **집단 지성 안전 그리드**: 주행 데이터가 누적되어 안전한 구역이 비취색(Cyan) 격자로 지도에 자동 렌더링됩니다.

3.  **실시간 기상 정보 (Open-Meteo API)**
    *   천안 지역의 현재 기온과 강수 여부를 실시간 API를 통해 수신합니다.
    *   강우/강설 시 자동으로 '기상 위험 모드'가 활성화되어 음성 안내를 제공합니다.

4.  **지오펜싱 검증 (Geofencing Logic)**
    *   **주차 확인**: 실제 주차장 좌표 데이터를 기반으로 사용자의 현재 위치가 반경 10m 이내인지 수학적으로 계산하여 검증합니다.
    *   **위험 알림**: 위험 구역 반경 50m 진입 시 즉각적인 음성 경고를 발령합니다.

5.  **PWA (Progressive Web App)**
    *   오프라인 캐싱 및 홈 화면 설치 기능을 지원하여 실제 앱처럼 사용 가능합니다.

## 🛠 최근 주요 개선 사항 (2026-05-06)

*   **Vercel 배포 안정화**: 중첩된 폴더 구조(`c-pm-safety`)에서 발생하는 빌드 및 CSS 깨짐 문제를 `vercel.json` 설정을 통해 완벽히 해결하였습니다.
*   **주행 중 UI(Dashboard) 강화**: 
    *   'LIVE RIDING' 애니메이션 인디케이터 추가로 주행 상태 가독성 향상.
    *   운행 종료 버튼 디자인 개선으로 오동작 방지 및 심미성 확보.
*   **Tailwind CSS v4 최적화**: 최신 테마 시스템과 Vite 플러그인을 적용하여 성능을 최적화하였습니다.

## 📁 파일 구조
*   `/c-pm-safety/src/App.jsx`: 메인 로직 및 UI 상태 관리
*   `/c-pm-safety/src/hooks/`: 데이터 통신(useSafeData), 위치(useGeolocation), 주행(useRideSession) 관련 커스텀 훅
*   `/c-pm-safety/src/utils/`: 거리 계산(Haversine), 물리 연동 등 핵심 알고리즘
*   `/c-pm-safety/src/data/`: 주차장 및 정적 위치 정보
